Немного о Api
Как оно работает?
Для начала загрузите все файлы на сервер в файл, а потом заполните все данные в config.php и config_for_request.php, после чего авторизуйте ваш сервер, за счёт скрипта в auth.php (Просто зайдите В этот файл, если вы всё правильно заполнили, страница выведет Json код, с 5 блоками информации), после авторизации желательно удалить этот файл!
Так же access_token живёт лишь один день, так что в access.php происходит "переавторизация" токена, его трогать не надо
Основной же запрос строится в amo.php, в чем его суть, там функция, которая содержит аргументы, все аргументы помещаются в массив и отправляются как значения полей в панель amoCRM, для создания запроса, просто вызовите функцию и заполните в ней аргументы 
Многие знают, что в АмоКрм каждое поле и воронка имеют свой уникальный id, для настройки этих id отредактируйте файл config_for_request.php, все значения id берутся именно оттуда
Если нужно отредактировать запрос, то измените массив $data в файле amo.php согласно Api документациям
